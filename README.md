# Rain Precipitation Prediction

Predict rain using ANN, CNN, and RNN.

## To run the model

```bash
git clone https://github.com/dharaneshwaran7/rain_prediction.git
cd rain_prediction
```
to run cnn model

```bash
python cnn.py
```

to run rnn model

```bash
python rnn.py
```

to run ann model

```bash
python ann.py
```
